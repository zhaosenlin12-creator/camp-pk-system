### 🔗 下载方法

#### 方法1：B站下载
1. 打开 https://www.bilibili.com
2. 搜索关键词：
   - 搞笑动物：搜"动物搞笑合集"
   - 电影片段：搜"经典电影片段 无字幕"
   - 动画场景：搜"动画搞笑配音素材"
   - 新闻播报：搜"新闻主播合集"
   - 广告片段：搜"搞笑广告合集"
3. 找到30秒-1分钟的片段
4. 复制视频链接到 https://xbeibeix.com/api/bilibili/ 下载
5. 重命名为上述文件名

#### 方法2：抖音下载
1. 打开抖音APP
2. 搜索相关内容
3. 复制链接到 https://www.douyin.wtf/ 下载
4. 重命名为上述文件名


#### 方法3：免费素材网站
- **Pexels**: https://www.pexels.com/videos/
- **Pixabay**: https://pixabay.com/videos/
- **Coverr**: https://coverr.co/

## ⚠️ 注意事项

1. **视频格式**: MP4 (H.264编码)
2. **视频时长**: 30秒-1分钟最佳
3. **分辨率**: 720p 或 1080p
4. **文件大小**: 单个文件不超过 20MB
5. **内容要求**: 
   - 配音视频最好是无对白或少对白的
   - 内容健康适合学生观看
