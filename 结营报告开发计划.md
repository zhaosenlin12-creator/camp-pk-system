## 数据结构设计

### reports 表
```json
{
  "id": 1,
  "student_id": 1,
  "class_id": 1,
  "report_code": "abc123",  // 唯一访问码
  "ai_message": "老师寄语内容...",
  "ai_prompt": "使用的提示词",
  "certificates": [1, 2],   // 关联的奖状ID
  "photos": [1, 2, 3],      // 关联的照片ID
  "status": "draft",        // draft/published
  "created_at": "2026-01-07T00:00:00Z",
  "published_at": null
}
```

### photos 表
```json
{
  "id": 1,
  "student_id": 1,
  "filename": "photo_1704672000000.jpg",
  "original_name": "活动照片.jpg",
  "created_at": "2026-01-07T00:00:00Z"
}
```

### certificates 表
```json
{
  "id": 1,
  "student_id": 1,        // 学员奖状
  "team_id": null,        // 或战队奖状
  "type": "first_prize",  // 奖状类型
  "title": "一等奖",
  "custom_text": "",      // 自定义文字
  "created_at": "2026-01-07T00:00:00Z"
}
```

### 奖状类型枚举
